通过调用 Ex35 容器的公共接口，对其进行适配，从而实现栈语义
Stack 不是 Container 的子类实现，而是对 Container 的接口约束、重映射
    像 hook 方法是给具体容器实现的，例如 Vector、List
    而 Stack 不是存储结构，只是限制接口使用方式

        ┌────────────────────┐
        │    Vector<T>       │
        └────────▲───────────┘
                 │ private inheritance
        ┌────────────────────┐
        │     Stack<T>       │
        ├────────────────────┤
        │ + push()           │
        │ + pop()            │
        │ + top()            │
        │ + size()           │
        │ + empty()          │
        │ + clear()          │
        └────────────────────┘
